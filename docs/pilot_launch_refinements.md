# Goodwill Circle Pilot Launch Refinements

## Launch Shape

Start with a controlled 30-day pilot, not a public launch. The first version should feel alive because real NGOs, colleges, and ambassadors are already using it.

Recommended pilot:

- 1 city or local zone
- 2-5 verified NGOs
- 1-2 colleges, NSS units, Rotaract clubs, or local communities
- 20-50 trained volunteers
- 100 curated real help requests or agenda opportunities

## Cold Start Plan

Seed only real activity. Do not create fake requests.

Initial sources:

- NGO partner beneficiary requests
- College community help requests
- Local volunteer organization needs
- NGO agenda items such as classes, mentoring, food drives, and elderly assistance
- Clearly labeled training or demo content for onboarding only

## Trust System

Use verification before scaling:

- User profiles can request verification.
- NGOs must be verified before posting NGO agenda items.
- Campaigns can be reviewed and marked verified.
- Abuse reports are stored for moderation.
- Verification approval happens through admin-only RPCs, not client-side updates.

Trust labels to show:

- Verified profile
- Verified NGO
- Verified campaign
- Pending review

## NGO Agenda Track

Keep NGO agenda separate from normal peer help requests because it has a different incentive model.

Agenda examples:

- Need one teacher for 20 weekend learners
- Need three volunteers for food distribution
- Need a health-awareness speaker
- Need mentors for college applications

Reward model:

- Volunteer connects to agenda
- NGO confirms completion
- Badge is awarded
- Certificate record is issued
- Impact and reputation increase

## Pilot Metrics

Track weekly:

| Metric | Pilot Target |
| --- | --- |
| Requests created | 50+ |
| NGO agenda items | 20+ |
| Fulfillment rate | >60% |
| Average response time | <48h |
| Returning users | >30% |
| Volunteer hours | Track |
| Certificates issued | Track |
| Abuse reports | Review all |

## Public Impact Report

At the end of the pilot, publish a small report:

- People helped
- Requests fulfilled
- NGO agenda completions
- Volunteer hours
- Certificates issued
- Campaign credits raised
- User and NGO feedback

This report becomes the proof for more NGOs, colleges, CSR teams, grants, and city expansion.
