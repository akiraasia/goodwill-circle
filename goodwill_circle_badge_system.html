Flutter / SVG Implementation Notes

flutter_svg Usage

Use SvgPicture.asset() for static badges. For animated unlocks, use Lottie with the SVG as the base frame. Recommended sizes: 48×48 (list), 64×64 (grid), 96×96 (profile), 128×128 (detail modal).

Locked State

Apply ColorFilter.matrix(grayscale) + 40% opacity overlay in Flutter for locked badges. SVG structure stays identical — only the rendering filter changes.

48px Rendering

At 48px, reduce fine detail: drop subtle opacity lines (parallels on globe, page lines on book). Keep only 1–2 key shapes per icon. All icons are designed with a 28px safe inner zone for legibility.
